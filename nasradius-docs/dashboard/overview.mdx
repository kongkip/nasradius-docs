## Overview

The Dashboard is your command center for monitoring your ISP business. It provides real-time insights into:

- Revenue metrics (daily, weekly, monthly, total)
- Customer statistics
- Online user counts
- Performance trends

## Key Components

### Stat Cards

The top section displays key metrics as interactive cards:

| Metric | Description | Trend |
|--------|-------------|-------|
| Daily Customers | New customers today | 30-day sparkline |
| Daily Revenue | Revenue collected today | 30-day sparkline |
| Weekly Revenue | This week's total revenue | 4-week sparkline |
| Total Customers | All registered customers | Cumulative trend |
| Total Revenue | All-time revenue | Cumulative trend |
| Online Customers | Currently connected users | 24-hour activity |

## Navigation

From the dashboard, you can quickly access:

- **Online Customers**: Click the "View" button on the Online Customers card
- **Settings**: Access via the sidebar
- **Other sections**: Use the sidebar navigation

## Understanding the Metrics

### Revenue Calculations

```
Daily Revenue = Sum of all payments received today
Weekly Revenue = Sum of payments from Monday to current day
Monthly Revenue = Sum of payments for the current month
Total Revenue = All-time sum of payments
```

### Customer Counts

```
Daily Customers = New registrations today
Total Customers = All registered accounts
Online Customers = Active sessions right now
```
