## Overview

Configure your SMS provider to enable customer notifications.

## Supported Providers

<CardGroup cols={3}>
  <Card title="AfricasTalking" icon="message">
    Popular African SMS gateway
  </Card>
  <Card title="CelcomAfrica" icon="message">
    East African SMS provider
  </Card>
  <Card title="MobiTech" icon="message">
    SMS with custom sender names
  </Card>
</CardGroup>

## Configuration Steps

<Steps>
  <Step title="Click Edit">
    Click the edit icon to modify settings
  </Step>
  <Step title="Select Provider">
    Choose your SMS provider from the dropdown
  </Step>
  <Step title="Enter Credentials">
    Fill in provider-specific credentials
  </Step>
  <Step title="Save">
    Click **Save Settings**
  </Step>
</Steps>

## Provider Credentials

<Tabs>
  <Tab title="AfricasTalking">
    | Field | Description |
    |-------|-------------|
    | Username | Your AfricasTalking username |
    | API Key | API key from dashboard |
  </Tab>
  <Tab title="CelcomAfrica">
    | Field | Description |
    |-------|-------------|
    | Partner ID | Your partner ID |
    | Shortcode | SMS shortcode |
    | API Key | API key |
  </Tab>
  <Tab title="MobiTech">
    | Field | Description |
    |-------|-------------|
    | Sender Name | Display name for SMS |
    | API Key | API key |
  </Tab>
</Tabs>

## Testing

After configuration, send a test notification to verify SMS delivery.
