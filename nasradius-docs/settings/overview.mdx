## Overview

The Settings section allows you to configure various aspects of your NasRadius installation.

## Accessing Settings

1. Click **Settings** in the sidebar
2. Navigate between tabs at the top
3. The current mode (Hotspot/PPPoE) affects available options

## Settings by Mode

| Setting | Hotspot | PPPoE |
|---------|---------|-------|
| General Settings | ✅ | ✅ |
| SMS Settings | ✅ | ✅ |
| Balance Reminders | ✅ | ✅ |
| NAS Devices | ✅ | ✅ |
| Payment Settings | ✅ | ❌ |

## Tab Navigation

Settings are organized into tabs. Click each tab to access different configuration areas. Your current tab selection is preserved in the URL.
