## Overview

The Router List shows all Mikrotik devices configured for your NasRadius installation.

## Accessing Router List

1. Click **Mikrotiks** in the sidebar
2. The dropdown expands with submenu
3. Click **Router List**

## Router Information

Each router displays:

| Field | Description |
|-------|-------------|
| Name | Router identifier |
| Location | Physical location |
| IP Address | NAS IP or Public IP |
| Status | Connection status |
| WebFig | Quick link to admin interface |

## Router Status

| Status | Meaning |
|--------|---------|
| Online | Responding to requests |
| Offline | Not reachable |
| Unknown | Status not determined |
