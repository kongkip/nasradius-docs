## Overview

Plan Management allows you to configure which internet plans can be purchased using Kopa.

## Enabling Plans for Kopa

<Steps>
  <Step title="Access Plan Management">
    Navigate to Kopa â†’ Plan Management tab
  </Step>
  <Step title="Select Plans">
    Choose which plans to enable for Kopa
  </Step>
  <Step title="Configure Terms">
    Set interest rates and payment periods if applicable
  </Step>
  <Step title="Save">
    Apply changes to the plans
  </Step>
</Steps>

## Plan Configuration

| Setting | Description |
|---------|-------------|
| Kopa Enabled | Allow this plan for BNPL |
| Interest Rate | Additional charge (if any) |
| Payment Period | Days to repay |
| Max Amount | Maximum Kopa limit |
