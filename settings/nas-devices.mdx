## Overview

NAS (Network Access Server) devices are your Mikrotik routers that handle user authentication.
Adding and Nas device here allows you to manage your mikrotik remotely using our wireguard VPN.
## NAS Configuration

Each NAS device stores:

| Field | Description |
|-------|-------------|
| Name | Device identifier |
| Location | Physical location |
| NAS IP | Internal IP address |
| Public IP | External IP address |
| WebFig URL | Router admin URL |

## Adding a NAS Device

<Steps>
  <Step title="Click Add NAS Device">
    Click the **Add NAS Device** button
  </Step>
  <Step title="Enter Details">
    Fill in device name, location, and IP addresses

    <Tip>
      To fill in the WebFig url, navigate to your mikrotik router and navigate to ip > routes. Then click on `wg-vpn` check the local address.
      If the local address is 10.0.0.5, then the webfig url is `https://r5.router.nasradius.com`
    </Tip>
    <Frame>
      <img src="/images/wg-vpn.png" alt="WebFig URL" />
    </Frame>
  </Step>
  <Step title="Save">
    Click **Create** to add the device
  </Step>
</Steps>

## Editing a NAS Device

1. Click the edit icon on the device card
2. Update the desired fields
3. Click **Update** to save changes

## Deleting a NAS Device

1. Click the delete icon on the device card
2. Confirm the deletion


## Best Practices

- Use descriptive names (e.g., "Office-Main-Router")
- Keep IP addresses updated
- Document WebFig URLs for quick access
- Include location for multi-site setups
