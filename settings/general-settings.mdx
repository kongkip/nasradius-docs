## Overview

General Settings allows you to configure basic information about your ISP.

## Configurable Fields

| Field | Description |
|-------|-------------|
| ISP Name | Your business name displayed throughout |
| Contact Phone | Customer support number |

## Editing Settings

<Steps>
  <Step title="Click Edit Icon">
    Click the pencil icon to enter edit mode
  </Step>
  <Step title="Update Fields">
    Modify ISP Name and/or Contact Phone
  </Step>
  <Step title="Save">
    Click **Save Settings** to apply changes
  </Step>
</Steps>

## Where Settings Are Used

| Location | Field Used |
|----------|------------|
| Browser Tab | ISP Name |
| Sidebar Header | ISP Name |
| SMS Notifications | ISP Name, Contact Phone |
| Customer Support | Contact Phone |

## Best Practices

<AccordionGroup>
  <Accordion icon="building" title="Use Official Name">
    Match your registered business name for consistency
  </Accordion>
  <Accordion icon="phone" title="Valid Phone">
    Ensure the contact number is always reachable
  </Accordion>
  <Accordion icon="text-width" title="Keep It Short">
    Shorter names display better in the interface
  </Accordion>
</AccordionGroup>
