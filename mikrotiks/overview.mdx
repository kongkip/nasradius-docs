## What You Can Do

- View all configured Mikrotik devices
- Access router WebFig interfaces
- Follow setup guides for Hotspot and PPPoE
- Download configuration scripts

## Prerequisites

Before configuring Mikrotik for NasRadius:
- Access to Mikrotik via Winbox or WebFig
- Your tenant hostname
- Admin privileges on the router
- Basic RouterOS knowledge

## Integration Points

| PPPoE | PPPoE Server |
