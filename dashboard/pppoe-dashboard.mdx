## Overview

The PPPoE Dashboard is designed for managing Point-to-Point Protocol over Ethernet subscribers, typically used for:
- Home internet connections
- Business internet services
- Fixed-line subscriber management

## PPPoE vs Hotspot Dashboard Differences

| Feature | PPPoE Dashboard | Hotspot Dashboard |
|---------|-----------------|-------------------|
| Authentication | Username/Password | Phone/Voucher |
| Session Type | Persistent connection | Time/Data limited |
| Payment Settings Tab | ❌ Not available | ✅ Available |
| SMS Usage Chart | ❌ Not shown | ✅ Available |
| Customer MAC Address | ❌ Not primary | ✅ Displayed |

## Understanding PPPoE Metrics

### Connection Status

PPPoE connections are typically:
- **Persistent**: Customers stay connected for extended periods
- **Billed monthly**: Usually on subscription basis
- **Profile-based**: Each customer has an assigned profile with speed limits

### Revenue Tracking

```
PPPoE Revenue = Monthly subscription fees + Overage charges
Typically: Prepaid monthly payments or postpaid billing
```
