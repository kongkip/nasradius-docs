## Overview

The Transactions page provides a complete log of all payment transactions, including both successful and pending payments.

## Accessing Transactions

Click **Transactions** in the sidebar navigation.

## Transaction List

| Column | Description |
|--------|-------------|
| # | Row number |
| Customer | Account number |
| Code | Transaction code |
| Phone | Payer's phone |
| Amount | Transaction amount |
| Status | Payment status |
| Date | Transaction timestamp |

- Reference transaction codes when handling disputes
- Monitor for failed transactions that need retry
