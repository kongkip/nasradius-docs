## Overview

Internet Plans are the foundation of your billing system. They define the service packages you offer to customers, including speeds, data limits, validity periods, and pricing.

## Accessing Plan Setup

1. Click **Plan Setup** in the sidebar
2. Ensure you're in the correct mode (Hotspot/PPPoE) using the toggle

<Frame>
  <img src="/images/plan-setup.png" alt="Plan Setup Page" />
</Frame>

## Plan List

The Plan Setup page shows all your internet plans:

| Column | Description |
|--------|-------------|
| # | Row number |
| Plan | Plan name |
| Code | Unique identifier |
| Unlimited | Whether data is unlimited |
| Data Bundle | Data limit in MB (if not unlimited) |
| Upload/Download | Speed limit |
| Validity | Duration in hours/days |
| Price | Cost in local currency |

## Creating a New Plan

<Steps>
  <Step title="Click Create Plan">
    Click the **Create Plan** button at the top
  </Step>
  <Step title="Enter Plan Details">
    Fill in the required fields
  </Step>
  <Step title="Save">
    Click **Submit** to create the plan
  </Step>
</Steps>

### Plan Fields

<ParamField path="name" type="string" required>
  Display name for the plan (e.g., "Daily Unlimited", "Weekly 5GB")
</ParamField>

<ParamField path="code" type="string" required>
  Unique identifier code. Auto-generated
</ParamField>

<ParamField path="isUnlimited" type="boolean" default="false">
  Toggle for unlimited data
</ParamField>

<ParamField path="dataBundle" type="number">
  Data limit in MB (only if not unlimited)
</ParamField>

<ParamField path="mikrotikRateLimit" type="string" required>
  Upload/Download speed limit in Mikrotik format
  
  Examples:
  - `5M/10M` = 5Mbps upload, 10Mbps download
  - `1M/2M` = 1Mbps upload, 2Mbps download
</ParamField>

<ParamField path="validityMinutes" type="number" required>
  Duration in minutes
  
  Examples:
  - 60 = 1 hour
  - 1440 = 1 day
  - 10080 = 1 week
</ParamField>

<ParamField path="price" type="number" required>
  Cost in your local currency
</ParamField>

## Editing a Plan

<Steps>
  <Step title="Find the Plan">
    Use search or scroll to find the plan
  </Step>
  <Step title="Click Edit Icon">
    Click the edit (pencil) icon in the Action column
  </Step>
  <Step title="Update Fields">
    Modify the desired fields
  </Step>
  <Step title="Save Changes">
    Click **Submit** to save
  </Step>
</Steps>

<Warning>
  Editing a plan does not affect existing active vouchers. Only new vouchers will use the updated settings.
</Warning>

## Deleting a Plan

<Steps>
  <Step title="Click Delete Icon">
    Click the delete (trash) icon on the plan row
  </Step>
  <Step title="Confirm Deletion">
    Click **Delete** in the confirmation dialog
  </Step>
</Steps>

<Note>
  Active vouchers using the deleted plan will continue to work until they expire.
</Note>

## Speed Format

Mikrotik Rate Limit format:

```
[upload_speed]/[download_speed]
```

Examples:
| Value | Upload | Download |
|-------|--------|----------|
| `1M/2M` | 1 Mbps | 2 Mbps |
| `5M/10M` | 5 Mbps | 10 Mbps |
| `10M/20M` | 10 Mbps | 20 Mbps |
| `512K/1M` | 512 Kbps | 1 Mbps |
