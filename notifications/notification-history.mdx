## Overview

The Notifications page displays all sent SMS notifications as cards showing:
- Customer name
- Message content
- Delivery status
- Send date
- Retry count

## Notification Cards

Each notification displays:

| Field | Description |
|-------|-------------|
| Customer | Recipient's name/account |
| Status | Delivery status |
| Retries | Number of retry attempts |
| Date | When sent |
| Message | SMS content |

## Viewing Full Messages

For long messages:
1. Click **Read More** to expand
2. Click **Show Less** to collapse

## Copying Messages

To copy a message:
1. Click the **Copy** button on the notification card
2. Message is copied to clipboard
3. Button shows "Copied" confirmation

## Pagination

Navigate through notifications using page numbers at the bottom. 10 notifications display per page.

## Searching History

Search by:
- Customer name
- Message content

## Delivery Statuses

| Status | Meaning |
|--------|---------|
| Sent | Successfully dispatched |
| Delivered | Confirmed received |
| Failed | Delivery unsuccessful |
| Pending | Awaiting delivery |

## Retry Information

The retry count shows how many times the system attempted delivery. High retry counts may indicate:
- Invalid phone number
- Network issues
- SMS provider problems
